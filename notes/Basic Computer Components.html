The computer receives input, processes with CPU and memory, and then sends results as output.

CPU

The brain of the PC, containing the circuitry needed to process input, store data, and output results. The CPU follows instructions of computer programs that explain how to process the data.

The Arithmetic Logic Unit (ALU) does the math and logic processing. The control unit interprets program instructions and is in charge of the process.

The CPU is an integrated circuit known as a chip

Computer Memory

When input devices send binary to the CPU, it immediately stores the data in memory. Since the CPU constantly uses data from memory, they're connected via the memory bus.

Types:

  1. Main memory (RAM):
    • only used for temporary storage of data
    • restarting a computer wipes RAM storage
    • there are two types: SRAM (static) and DRAM (dynamic)
  2. External/Secondary Memory:
    • hard drive
    • stores long-term data
    • takes longer to load, but much larger
  3. Read-Only Memory:
    • can only be read from, only used for integrated chip on the hardware that is written beforehand
    • used for basics of PC, such as identifying/controlling hardwares

Secondary Memory

Auxiliary storage, used for long-term retrieval. PCs come with a hard drive, which is permanently attached. There are also usually ports for external storage such as USB Drives and SD Cards. These can be used to transfer between one computer to another.

There are many technologies used to implement Secondary Storage Devices, such as flash memory, optical discs, and magnetic disks.

Types:

  1. Solid State Drive (SSD):
    • stores things digitally
  2. Hard Disk Drive (HDD):
    • stores on physical disk
